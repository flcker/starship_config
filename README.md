## Nerd Font

Nerd Fonts 是一系列经过“打补丁”（Patched）的字体，它们在原有优秀字体（如 Fira Code, JetBrains Mono）的基础上，加入了数以千计的图标（如 Font Awesome, Devicons, Octicons 等）。
这对于提升开发环境、终端（Terminal）以及代码编辑器的视觉美感至关重要，特别是当你使用 Powerlevel10k、Starship 或 Oh My Zsh 时。

Starship 依赖Nerd Font, 这是非常重要的依赖，必须安装；需要设置终端字体为Nerd Font, 否则会显示异常。

WebSites: https://www.nerdfonts.com/
